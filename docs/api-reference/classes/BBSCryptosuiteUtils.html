    • BBSCryptosuiteUtils

    Class BBSCryptosuiteUtils

    Minimal BBS utility methods ported from legacy for working with Data Integrity BBS (bbs-2023) base and derived proof value encoding.

    Notes:

    • This module focuses on serialization/parsing helpers used by callers to pack/unpack proof values. It does not perform signing or verification.
    • All methods operate on Uint8Array inputs and return multibase strings (base64url with 'u' prefix) where applicable to match the spec.
