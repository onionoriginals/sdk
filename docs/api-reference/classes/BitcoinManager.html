    Class BitcoinManager

    Manages Bitcoin/Ordinals operations including inscription creation, transfer, and DID validation on the Bitcoin blockchain.

    Requires an OrdinalsProvider to be configured for all Bitcoin operations. Optionally accepts a FeeOracleAdapter for dynamic fee estimation.

    Example

    const sdk = OriginalsSDK.create({
      network: 'regtest',
      ordinalsProvider: new OrdMockProvider(),
    });
    const inscription = await sdk.bitcoin.inscribeData(
      { hello: 'world' },
      'application/json'
    );
