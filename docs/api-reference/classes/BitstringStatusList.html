    Class BitstringStatusList

    W3C Bitstring Status List implementation.

    A compressed bitstring where each bit position represents the status of a credential. A bit value of 1 means the status purpose applies (e.g. revoked or suspended). A bit value of 0 means it does not.

    See

    https://www.w3.org/TR/vc-bitstring-status-list/
